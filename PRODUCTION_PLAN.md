# Bajaj Sports Production Roadmap

## Phase 1: Digital First (In Progress)
- ✅ **WhatsApp Integration:** Direct quote sharing from UI to client.
- ✅ **Cloud Access:** Secure Tailscale access for the team.
- ✅ **High-Fidelity OCR:** Mistral and Sarvam AI integration for 100% accuracy.
- 🚧 **Sarvam AI Deployment:** Running the full 1000+ batch using Sarvam Vision.

## Phase 2: Internal Ops
- [ ] **Data Scrubbing:** Final catalog cleanup using the Sarvam spatial output.
- [ ] **Inventory Sync:** Let owner update stock/prices manually via UI.
- [ ] **PDF Archive:** Auto-naming and cloud backup for every quote generated.
- [ ] **Multi-user:** Simple login for different staff members.

## Phase 3: Intelligence
- [ ] **Trend Alerts:** Notify owner if a specific item is being quoted frequently.
- [ ] **Auto-Follow-up:** Reminder to check with client 3 days after quote.
