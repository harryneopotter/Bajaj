# Bajaj Sports Project - Directory & File Inventory

This directory contains the Quote Vault and Quotation Generator system. 

## 🏗️ Core Application (Production)
| File / Directory | Purpose | Status |
| :--- | :--- | :--- |
| `quotegen/` | **Main Application Folder**. Contains the FastAPI app, SQLite database, and frontend assets. | **ACTIVE** |
| `quotegen/main.py` | The main server script. Handles UI, API endpoints (Client/Product search), and PDF generation. | **ACTIVE** |
| `quotegen/quotes.db` | SQLite database storing all new quotations created via the web interface. | **ACTIVE** |
| `quotegen/static/` | Folder for static assets (logo, broken scans for audit, CSS/JS). | **ACTIVE** |
| `quotegen/venv/` | Python virtual environment with all required dependencies (FastAPI, WeasyPrint, Sarvam, etc.). | **ACTIVE** |

## 🧠 Data & Analysis (Production)
| File / Directory | Purpose | Status |
| :--- | :--- | :--- |
| `analysis/` | Folder containing processed JSON data extracted from historical PDFs. | **ACTIVE** |
| `analysis/customer_purchases.json` | **Master Client List**. Stores address, phone, and purchase history for 62 high-fidelity clients. | **ACTIVE** |
| `analysis/clean_catalog.json` | **Master Product Catalog**. Stores 2,258 products with historical pricing tiers. | **ACTIVE** |
| `analysis/pdf_client_mapping.json` | Links each source PDF filename to the client name extracted from it. | **ACTIVE** |
| `data/pdf/` | Storage for the original 283+ source PDF documents. | **ACTIVE** |
| `extracted/sarvam/` | High-fidelity HTML reconstructions of the PDFs generated by Sarvam AI. | **ACTIVE** |

## ⚙️ Automation & Processing Scripts
| File | Purpose | Status |
| :--- | :--- | :--- |
| `continuous_ingest.py` | Background service that auto-processes new PDFs through Sarvam and syncs to the DB. | **ACTIVE** |
| `sarvam_parse.py` | Core engine for interacting with Sarvam AI Vision API. | **ACTIVE** |
| `final_production_scrub.py` | Advanced cleaning script used to purge noise (GST, IRNs) and merge duplicate clients. | **ACTIVE** |
| `parse_html.py` | Logic to convert Sarvam HTML output into the JSON catalog format. | **ACTIVE** |
| `mistral_parse.py` | Secondary engine for processing via Mistral OCR. | **STANDBY** |
| `deep_pdf_parse.py` | Legacy direct PDF text extractor (before Sarvam integration). | **OUTDATED** |
| `deep_extract.py` | Early regex-based text extractor. | **OUTDATED** |
| `invoice_parser.py` | Initial MVP skeleton parser. | **OUTDATED** |
